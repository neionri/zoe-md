# Security Policy 🛡️🔐

Zoe Neural Project sangat menghargai integritas data dan privasi. Jika Anda menemukan celah keamanan pada syaraf Zoe, harap ikuti prosedur di bawah ini:

## Melaporkan Vulnerability
Jangan laporkan celah keamanan melalui menu "Issues" publik. Silakan hubungi pemilik repositori (Boss Neionri) melalui jalur pribadi yang tersedia atau ajukan laporan melalui fitur **Security Advisories** di GitHub jika Anda memiliki akses.

## Cakupan Keamanan
Kami memprioritaskan perbaikan untuk:
- Kebocoran API Key (Groq, MongoDB).
- Bypass pada sistem perintah Bouncer.
- Eksploitasi database melalui message handler.

## Komitmen Kami
Kami akan merespon laporan Anda dalam waktu maksimal 48 jam dan bekerja sama untuk menambal celah tersebut sebelum dipublikasikan ke publik.
